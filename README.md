## Desafio 👾

Utilizando **Nextjs**, siga os passos abaixo 🎯

- **Replicar a tela de cadastro da Mais Retorno**

  - Ao clicar em criar a conta, deve ocorrer a validação dos campos.
  - Os campos devem ter a seguinte validação
    - Nenhum campo pode estár vázio
    - O email deve ser um email válido
    - A senha deve ter ao menos 8 caracteres, conter ao menos uma letra e um número
  - Os botões não precisam ter ações, exceto o botão "CRIAR CONTA".
  - Ao clicar no botão **CRIAR CONTA**, se todos os campos forem válidos, direcione o usuário para tela de conta criada com sucesso

- **Tela de conta criada com sucesso**

  - A tela deve receber o nome do usuário digitado na tela anterior e exibir onde está indicado no prótotipo "**NOMEDIGITADO**"
  - O botão não precisa ter ação

- Envie o link do repositório com a resolução desse desafio para matheus.bueno@maisretorno.com
